# Iannix Library

```python
from iannix import *
```

The iannix library supports communication with [IanniX](https://www.iannix.org/en/) through the [OSC](../../osc/index.md) protocol.

| Contents | Description |
|---|---|
| [IannixIn](iannixin/index.md) | Receive messages from a running IanniX application over OSC. |
| [IannixOut](iannixout/index.md) | Send messages to a running IanniX application over OSC. |
