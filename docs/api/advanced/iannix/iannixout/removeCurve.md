# removeCurve()

Remove a curve from the IanniX score.

```python
iannixout.removeCurve(curveID)
```

## Parameters

```python
iannixout.removeCurve(curveID)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `curveID` | `int or str` | _required_ | The ID of the curve to remove. |
