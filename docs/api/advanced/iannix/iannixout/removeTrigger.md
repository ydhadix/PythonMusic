# removeTrigger()

Remove a trigger from the IanniX score.

```python
iannixout.removeTrigger(triggerID)
```

## Parameters

```python
iannixout.removeTrigger(triggerID)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `triggerID` | `int or str` | _required_ | The ID of the trigger to remove. |
