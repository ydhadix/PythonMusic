# removeCursor()

Remove a cursor from the IanniX score.

```python
iannixout.removeCursor(cursorID)
```

## Parameters

```python
iannixout.removeCursor(cursorID)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `cursorID` | `int or str` | _required_ | The ID of the cursor to remove. |
