# Text

These text objects can be used to present plain text and interactive text fields.

| Contents | Description |
|---|---|
| [Label](label/index.md) | A label that shows a line of text. |
| [TextField](textfield/index.md) | A single-line text field the user can type into. |
| [TextArea](textarea/index.md) | A multi-line text area the user can type into. |
| [Font](font/index.md) | A text font with a name, style, and size. |
