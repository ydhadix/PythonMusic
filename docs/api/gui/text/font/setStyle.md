# setStyle()

Set the font's style.

```python
font.setStyle(style)
```

## Parameters

```python
font.setStyle(style)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `style` | `tuple` | _required_ | The new text style, one of Font.PLAIN, Font.BOLD, Font.ITALIC, or Font.BOLDITALIC. |
