# setBackgroundColor()

Set the label's background [Color](../../color/index.md).

## Parameters

Once an object `label` has been created, you can use the following functions:

```python
label.setBackgroundColor()
```

```python
label.setBackgroundColor(color)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `color` | `Color` | `None` | The new background color. If omitted, a color-selection dialog opens. |
