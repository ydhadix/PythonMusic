# Controls

These controls are used to present information and let the user visually control a program and add input.

| Contents | Description |
|---|---|
| [Button](button/index.md) | A clickable button. |
| [CheckBox](checkbox/index.md) | A labeled box that can be checked and unchecked. |
| [Slider](slider/index.md) | A vertical or horizontal slider that can be dragged to choose a value. |
| [DropDownList](dropdownlist/index.md) | A drop-down list that items can be picked from. |
| [Menu](menu/index.md) | A menu for a Display's menu bar or pop-up menu. |
| [HFader](hfader/index.md) | A horizontal fader that can be dragged left and right. |
| [VFader](vfader/index.md) | A vertical fader that can be dragged up and down. |
| [Rotary](rotary/index.md) | A rotary knob that can be turned. |
| [Push](push/index.md) | A push-and-hold button. |
| [Toggle](toggle/index.md) | A button that turns on and off with each click. |
| [XYPad](xypad/index.md) | A trackpad that can click and drag to choose a 2-dimensional value. |
