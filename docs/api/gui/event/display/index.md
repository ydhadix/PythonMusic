# Display Events

In addition to [mouse](../mouse/index.md) and [keyboard](../keyboard/index.md) events, [Display](../../display/index.md) objects also have an [onClose()](../../display/onClose.md) event handling function.

| Function | Description |
|---|---|
| [`display.onClose(action)`](../../display/onClose.md) | Set up a function to call right before the display closes. |
