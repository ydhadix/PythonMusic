# Display Events

In addition to [mouse](../mouse/index.md) and [keyboard](../keyboard/index.md) events, [Display](../../display/index.md) objects also have an [onClose()](../../common/events/display/onClose.md) event handling function.

| Function | Description |
|---|---|
| [`onClose(action)`](../../common/events/display/onClose.md) | Register a function to call right before the display closes. |
