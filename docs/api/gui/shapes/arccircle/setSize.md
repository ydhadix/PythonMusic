# setSize()

Set the arc-circle's size.

```python
arccircle.setSize(width, height)
```

Its width and height are always the same; if they differ, the width is used for both.

## Parameters

```python
arccircle.setSize(width, height)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `width` | `int or float` | _required_ | The new diameter, in pixels. |
| `height` | `int or float` | _required_ | The new diameter, in pixels (should match the width). |
