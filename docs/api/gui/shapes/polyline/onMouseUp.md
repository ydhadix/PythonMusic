# onMouseUp()

Set up a function to call when the mouse button is released over this object.

```python
polyline.onMouseUp(action)
```

## Parameters

```python
polyline.onMouseUp(action)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `action` | `function` | _required_ | The function to call; it receives two parameters, x and y, giving the mouse position in display coordinates. |
