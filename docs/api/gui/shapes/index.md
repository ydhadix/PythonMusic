# Shapes

Shapes are simple, geometric graphics objects that can be added to a [Display](../display/index.md).  These objects may have a specified color, thickness, and may be filled or not.

| Contents | Description |
|---|---|
| [Rectangle](rectangle/index.md) | A rectangle drawn between two opposite corners. |
| [Oval](oval/index.md) | An oval drawn between two opposite corners. |
| [Circle](circle/index.md) | A circle drawn around a center point. |
| [Point](point/index.md) | A single point. |
| [Arc](arc/index.md) | A radial slice of an oval. |
| [ArcCircle](arccircle/index.md) | A radial slice of a circle. |
| [Line](line/index.md) | A line drawn between two endpoints. |
| [Polyline](polyline/index.md) | Connected lines drawn between a series of endpoints. |
| [Polygon](polygon/index.md) | A connected shape drawn from a series of endpoints. |
