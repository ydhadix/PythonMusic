# onMouseDrag()

Set up a function to call when the mouse is dragged over this object.

```python
rectangle.onMouseDrag(action)
```

A drag is moving the mouse while a button is held down.

## Parameters

```python
rectangle.onMouseDrag(action)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `action` | `function` | _required_ | The function to call; it receives two parameters, x and y, giving the mouse position in display coordinates. |
