# getCenter()

Return the object's center point.

If the object is a [Display](../../display/index.md), this returns the center point of the display's canvas.

## Parameters

Once an object `item` has been created, you can use the following function:

```python
item.getCenter()
```

## Returns

`return centerX, centerY`

| Value | Type | Description |
|---|---|---|
| centerX | `int or float` | The horizontal position of the center, in pixels. |
| centerY | `int or float` | The vertical position of the center, in pixels. |
