# getCenterY()

Return the object's vertical center.

If the object is a [Display](../../display/index.md), this returns the vertical center of the display's canvas.

## Parameters

Once an object `item` has been created, you can use the following function:

```python
item.getCenterY()
```

## Returns

`return y`

| Value | Type | Description |
|---|---|---|
| y | `int or float` | The vertical center, in pixels. |
