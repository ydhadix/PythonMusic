# setY()

Set the object's vertical position.

If the object is a [Display](../../display/index.md), this sets the display's vertical position on the screen.

## Parameters

Once an object `item` has been created, you can use the following function:

```python
item.setY(y)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `y` | `int or float` | _required_ | The new vertical position, in pixels. |
