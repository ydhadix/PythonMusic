# setColor()

Set the object's [Color](../../color/index.md).

## Parameters

Once an object `item` has been created, you can use the following functions:

```python
item.setColor()
```

```python
item.setColor(color)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `color` | `Color` | `None` | The new color. If omitted, a color-selection dialog opens. |
