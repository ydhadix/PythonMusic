# getGroup()

Return the [Group](../../group/index.md) this object belongs to, if any..

## Parameters

Once an object `item` has been created, you can use the following function:

```python
item.getGroup()
```

## Returns

`return group`

| Value | Type | Description |
|---|---|---|
| group | `Group` | The group holding this object, or `None` if it is not in a group. |
