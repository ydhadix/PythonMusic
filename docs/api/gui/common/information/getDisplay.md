# getDisplay()

Return the [Display](../../display/index.md) this object is on.

## Parameters

Once an object `item` has been created, you can use the following function:

```python
item.getDisplay()
```

## Returns

`return display`

| Value | Type | Description |
|---|---|---|
| display | `Display` | The display showing this object, or `None` if it is not on a display. |
