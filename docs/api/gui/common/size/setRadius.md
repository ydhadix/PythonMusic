# setRadius()

Set the circle's radius.

**NOTE:** setRadius() is only available for [Circle](../../shapes/circle/index.md) and [ArcCircle](../../shapes/arccircle/index.md) objects.

## Parameters

Once an object `circle` has been created, you can use the following function:

```python
circle.setRadius(radius)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `radius` | `int or float` | _required_ | The new radius, in pixels. |
