# setSize()

Set the object's width and height.

## Parameters

Once an object `item` has been created, you can use the following function:

```python
item.setSize(width, height)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `width` | `int or float` | _required_ | The new width, in pixels. |
| `height` | `int or float` | _required_ | The new height, in pixels. |
