# getLength()

Return the line's length.

**NOTE:** getLength() is only available for [Line](../../shapes/line/index.md) objects.

## Parameters

Once an object `line` has been created, you can use the following function:

```python
line.getLength()
```

## Returns

`return length`

| Value | Type | Description |
|---|---|---|
| length | `int or float` | The distance between the line's two ends, in pixels. |
