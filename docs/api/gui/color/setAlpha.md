# setAlpha()

Set the alpha (transparency) component of the color.

```python
color.setAlpha(alpha)
```

## Parameters

```python
color.setAlpha(alpha)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `alpha` | `int` | _required_ | The new opacity, from 0 (fully transparent) to 255 (fully opaque). |
