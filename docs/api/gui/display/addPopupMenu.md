# addPopupMenu()

Add a pop-up (right-click) menu to the display.

```python
display.addPopupMenu(menu)
```

## Parameters

```python
display.addPopupMenu(menu)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `menu` | `Menu` | _required_ | The menu to show when the user right-clicks the display. |
