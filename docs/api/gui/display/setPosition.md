# setPosition()

Set the display's position on the screen.

```python
display.setPosition(x, y)
```

The screen origin (0, 0) is at the top-left.

## Parameters

```python
display.setPosition(x, y)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `x` | `int or float` | _required_ | The new horizontal position of the window's top-left corner, in pixels. |
| `y` | `int or float` | _required_ | The new vertical position of the window's top-left corner, in pixels. |
