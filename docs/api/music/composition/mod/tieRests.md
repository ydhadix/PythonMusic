# tieRests()

Join neighboring rests into one longer rest, in place.

```python
Mod.tieRests(material)
```

This lowers the note count.

## Parameters

```python
Mod.tieRests(material)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `material` | `Phrase, Part, or Score` | _required_ | The music to change. |
