# Composition

The music library offers the following classes to assist in [algorithmic music composition](http://en.wikipedia.org/wiki/Algorithmic_composition). These classes can be used in arbitrary programs to generate variations of musical material for inspiration and/or for [performance](../performance/index.md).

| Contents | Description |
|---|---|
| [Mod](mod/index.md) | A collection of functions to modify musical material. |
| [View](view/index.md) | Tools to visualize musical material. |
| [Read](read/index.md) | Read musical material from MIDI files. |
| [Write](write/index.md) | Write musical material to MIDI files. |
