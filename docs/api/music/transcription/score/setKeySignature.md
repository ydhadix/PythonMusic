# setKeySignature()

Set the score's key signature.

```python
score.setKeySignature(signature)
```

## Parameters

```python
score.setKeySignature(signature)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `signature` | `int` | _required_ | The key signature: 0 is the key of C, a positive number counts sharps, and a negative number counts flats. |
