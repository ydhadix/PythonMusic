# setPan()

Set the note's stereo position.

```python
note.setPan(pan)
```

## Parameters

```python
note.setPan(pan)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `pan` | `int or float` | _required_ | The new stereo position, from 0.0 (left) through 0.5 (center) to 1.0 (right). |
