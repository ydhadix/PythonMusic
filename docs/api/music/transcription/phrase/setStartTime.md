# setStartTime()

Set when the phrase starts.

```python
phrase.setStartTime(startTime)
```

## Parameters

```python
phrase.setStartTime(startTime)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `startTime` | `int or float` | _required_ | The new start time, in beats, where 0.0 is the start of the piece. |
