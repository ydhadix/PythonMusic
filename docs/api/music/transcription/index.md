# Transcription

The music library offers the following classes for music transcription. These classes allow the creation of arbitrary musical scores that can be [played](../play/index.md), [modified](../composition/mod/index.md), [viewed](../composition/view/index.md), and [saved](../composition/write/index.md) as a MIDI file.

| Contents | Description |
|---|---|
| [Note](note/index.md) | A single musical event, with pitch and duration. |
| [Phrase](phrase/index.md) | A sequence of notes. |
| [Part](part/index.md) | A set of phrases played with a single instrument. |
| [Score](score/index.md) | A set of parts played in parallel. |
