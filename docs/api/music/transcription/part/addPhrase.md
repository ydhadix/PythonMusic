# addPhrase()

Add a phrase to the part.

```python
part.addPhrase(phrase)
```

If the phrase has no set start time, it is placed at the end of the part.

## Parameters

```python
part.addPhrase(phrase)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `phrase` | `Phrase` | _required_ | The phrase to add. |
