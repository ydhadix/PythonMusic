# Performance

The music library offers the following classes for **music performance**. These are intended to be used with arbitrary [GUI instruments](../../gui/index.md), [MIDI instruments](../../midi/index.md), or [OSC instruments](../../osc/index.md) for real-time performance and/or for [live coding](http://en.wikipedia.org/wiki/Live_coding).

| Contents | Description |
|---|---|
| [AudioSample](audiosample/index.md) | A sound loaded from an audio file. |
| [Envelope](envelope/index.md) | Shape and AudioSample's volume over time. |
| [MidiSequence](midisequence/index.md) | Play midi files live. |
| [Metronome](metronome/index.md) | Synchronize music and function calls. |
