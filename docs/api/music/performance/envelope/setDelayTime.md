# setDelayTime()

Set the envelope's delay time.

```python
envelope.setDelayTime(delayTime)
```

## Parameters

```python
envelope.setDelayTime(delayTime)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `delayTime` | `int` | _required_ | How long to take reaching the sustain volume, in milliseconds after the last attack time. |
