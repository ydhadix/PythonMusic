# setReleaseTime()

Set the envelope's release time.

```python
envelope.setReleaseTime(releaseTime)
```

## Parameters

```python
envelope.setReleaseTime(releaseTime)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `releaseTime` | `int` | _required_ | How long the sound takes to fade to silence after it ends, in milliseconds. |
