# setSustainVolume()

Set the envelope's sustain volume.

```python
envelope.setSustainVolume(sustainVolume)
```

## Parameters

```python
envelope.setSustainVolume(sustainVolume)
```

| Parameter | Type | Default | Description |
|---|---|---|---|
| `sustainVolume` | `float` | _required_ | The volume held through the body of the sound, from 0.0 to 1.0. |
