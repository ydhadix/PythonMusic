# Panning Constants

Panning (or panoramic) constants are represented in the music library using real numbers (floats) from 0.0 (left) to 1.0 (right). Given that human ears are very sensitive to placement of sound in a stereo field, using floats allows for much greater detail than using integers.

The music library defines the following panning constants:

| Constant | Value |
|---|---|
| `PAN_LEFT` | `0.0` |
| `PAN_CENTER` | `0.5` |
| `PAN_RIGHT` | `1.0` |
