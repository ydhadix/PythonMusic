# Timer Library

```python
from timer import *
```

The **timer library** supports scheduling tasks (e.g., animations).

| Contents | Description |
|---|---|
| [Automate](automate/index.md) | Schedule many functions to run together at a fixed interval. |
| [Timer](timer/index.md) | Schedule a function to run after a delay, or to repeat at a fixed interval. |
| [LinearRamp](linearramp/index.md) | Slide a value smoothly from one number to another over time, calling a function as it changes. |
| [OscillatorTimer](oscillatortimer/index.md) | Call a function over and over with a value that oscillates between two bounds. |
| [EnvelopeTimer](envelopetimer/index.md) | Call a function with a series of values, each delivered at its own time. |
