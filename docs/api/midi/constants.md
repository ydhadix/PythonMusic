# MIDI Event Constants

Useful constants for assigning MIDI `eventType` to [MidiIn.onInput()](midiin/onInput.md) and [MidiOut.sendMidiMessage()](midiout/sendMidiMessage.md).

| Name | Value |
|---|---|
| `ALL_EVENTS` | `-1` |
| `NOTE_ON` | `144` |
| `NOTE_OFF` | `128` |
| `SET_INSTRUMENT` | `192` |
| `CONTROL_CHANGE` | `176` |
| `PITCH_BEND` | `224` |
| `AFTERTOUCH` | `208` |
| `POLYTOUCH` | `160` |
