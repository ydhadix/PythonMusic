# Reference

PythonMusic provides libraries for music making, image manipulation, building graphics interfaces, and connecting to external devices — such as digital pianos, smartphones, and tablets.

| Contents | Description |
|---|---|
| [Music Library](music/index.md) | Tools for creating and playing music. |
| [GUI Library](gui/index.md) | Tools for creating visuals and Graphical User Interfaces. |
| [Mapping Values](utilities/index.md) | Useful functions for mapping values between ranges. |
| [MIDI Library](midi/index.md) | Supports communication with MIDI-enabled devices. |
| [OSC Library](osc/index.md) | Supports communication with OSC-enabled devices. |
| [Timer Library](timer/index.md) | Tools for automation and scheduling function calls.  |
| [Advanced](advanced/index.md) | Libraries for advanced topics, like [Zipf metrics](advanced/zipf/index.md) and [Markov models](advanced/markov/index.md). |
